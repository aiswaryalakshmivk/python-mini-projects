# Python Mini Projects

This repository contains beginner-friendly Python mini programs for logic building, coding practice, and problem solving.

## Files Included

- `palindrome.py` – Check whether a string is a palindrome
- `fibonacci.py` – Generate Fibonacci series
- `armstrong.py` – Check whether a number is an Armstrong number
- `anagram.py` – Check whether two strings are anagrams
- `email_sort.py` – Sort email addresses alphabetically
- `recursion_example.py` – Example of recursion in Python

## Skills Used

- Python basics
- Conditional statements
- Loops
- Functions
- String operations
- Recursion

## Purpose

To strengthen Python programming skills and improve problem-solving ability.
